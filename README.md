## txt-file-editor

### Description

This project contain txt file editor written in C programming language.

### Installation

1. Clone repository from server.

  ```
  git clone https://gitlab.com/mapistarts/txt-file-editor.git
  ```

* Compile file
  ```
  clang main.c
  ```
  or
  ```
  gcc main.c
  ```
